## 🚀 Projects

### 💼 Company & Main Direction

- 🏢 **[TwargaOps](https://twargaops.com)** - My company and umbrella for open-source infrastructure tools, cybersecurity labs, cloud systems, and DevSecOps education.
- 🌐 **[Twarga.tech](https://twarga.tech)** - My personal blog and build log.
- 🍞 **[Yeast](https://github.com/Twarga/yeast)** - Linux-first local VM orchestrator built with Go, QEMU/KVM, and cloud-init. The infrastructure engine behind TwargaOps.
