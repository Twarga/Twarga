## About Me

I'm Youness Touzani, also known as **Twarga**.

I'm a DevSecOps-focused builder from Morocco. I care about Linux, infrastructure, automation, cybersecurity, open source, and practical education.

I started by learning programming, networking, Linux, and game development when I was younger. Over time, that path turned into a bigger goal: building useful systems, teaching what I learn, and creating tools that help people practice real infrastructure and security skills.

I use AI-assisted development tools, but I care deeply about architecture, product direction, debugging, testing, and understanding how systems work. Code is easier to generate now, but clear engineering thinking, responsibility, and taste still matter.

---

## Why I Build

One of my strongest motivations is simple: **I do not want to be useless.**

I do not mean that only in a career sense. I mean it as a way of living. I do not want to pass through life only consuming what other people created, watching from the side, or waiting for someone else to give me direction.

I want to understand things deeply. I want to build systems with my own hands, even when the path is messy. I want to become someone who can help, teach, repair, create, and carry responsibility.

Technology became one of the places where I can practice that. Linux, infrastructure, cybersecurity, open source, and education are not just topics for me. They are ways to turn curiosity into skill, skill into useful tools, and useful tools into opportunities for other people.

**TwargaOps comes from that idea.** It is my attempt to build something useful, document the path, and create tools and learning spaces that help others become more capable too.

---

## What I'm Building

I'm building **TwargaOps**, an umbrella company for open-source infrastructure tools, cybersecurity labs, education, and hosted systems.

The idea is simple:

```text
Open-source tools
  -> public trust
  -> practical labs
  -> courses and mentoring
  -> freelance / consulting
  -> hosted infrastructure through Twarga Cloud
```

The core tools are open source when possible. The business comes from the ecosystem around them: hosting, support, courses, mentoring, and implementation work.

## Featured Projects

### TwargaOps Direction

- **[TwargaOps](https://twargaops.com)**  
  My umbrella company for open-source infrastructure tools, cybersecurity labs, cloud systems, and practical DevSecOps education.

- **[Yeast](https://github.com/Twarga/yeast)**  
  Linux-first local VM orchestrator built with Go, QEMU/KVM, and cloud-init. The infrastructure engine behind TwargaOps.

- **LabsBackery**  
  Cybersecurity lab platform built on top of Yeast. The goal is to build, run, reset, and monitor multi-VM security labs with less setup pain.

- **Yeast MCP**  
  Planned AI control layer for Yeast-managed VMs, so agents can inspect, configure, test, and run real Linux workflows.

- **Twarga Cloud**  
  Planned hosted version of the TwargaOps ecosystem for people who want managed infrastructure instead of self-hosting.
